Supplier Aging EXE (Rowad Modern Engineering) tool.

This executable prompts the user to select 2 excel files as follows:
    1- Excel file containing a database for project name, projects manager, cost center (Default file attached: "General Categorization VBA.xlsm" in "Input Files" folder) 
    2- Excel file containing the supplier aging data in a particlar format as shown (Default file attached: "SWD_Supplier_Aging___RME_120723.xlsb")
